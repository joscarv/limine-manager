# limine-manager 1.4.1

Version 1.4.1 improves encrypted-root discovery and the accuracy of the `status` health report.

## Highlights

- Uses `cryptsetup status` to resolve the physical device behind a dm-crypt mapper.
- Reports the backing device, LUKS UUID, and PARTUUID.
- Removes quotes around paths in human-readable output.
- Separates the normalized root device from the Btrfs subvolume.
- Adds a degraded health state for incomplete encryption metadata.
- Reports configuration synchronization separately from system health.

The configuration schema remains version 1.
