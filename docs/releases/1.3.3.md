# limine-manager 1.3.3

Version 1.3.3 hardens encrypted-root validation for Archinstall and other valid device-reference styles.

## Highlights

- Supports `cryptdevice=PARTUUID=<partuuid>:<mapper>`.
- Compares the encrypted root using stable LUKS UUID, partition PARTUUID, canonical backing-device path, or mapper name.
- Records the PARTUUID of the encrypted backing partition during discovery.
- Includes a regression test for Archinstall with UKI, Btrfs, LUKS, the traditional `encrypt` hook, and a `cryptdevice=PARTUUID=` command line.

The configuration schema remains version 1.
