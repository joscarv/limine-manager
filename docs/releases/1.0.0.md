# limine-manager 1.0.0

This is the first stable release. It freezes configuration schema 1, the public CLI, and exit-code behavior for the 1.x series.

## Stable interfaces

- Configuration schema: `1`
- Exit codes: `0`, `1`, `2`, and `3`
- Managed backup naming: `limine.conf.bak.*`
- Core commands from `check-config` through `prune-backups`

## Upgrade

Version 0.9 configuration files remain valid. Run `limine-manager check-config`, then `validate` and `dry-run` before the first privileged `apply`.

## Known scope

The release targets Arch Linux systems using Limine, Btrfs, Snapper, and the Linux boot protocol. Other distributions and filesystem layouts are not yet declared supported.
