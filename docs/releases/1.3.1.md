# limine-manager 1.3.1

Version 1.3.1 expands boot-layout compatibility for Arch Linux installations created by `archinstall` and other tools that place Limine and kernel artifacts in EFI subdirectories.

## Highlights

- Recursive discovery under the boot mount.
- Automatic fallback from `/boot/limine.conf` to `/boot/EFI/BOOT/limine.conf`.
- Unified Kernel Image detection (`/boot/EFI/Linux/arch-<pkgbase>.efi`).
- EFI protocol rendering for UKIs, including command-line overrides used by snapshot entries.
- Correct preservation of nested paths in generated Limine configuration.
- Standalone initramfs and microcode are optional when they are embedded in a UKI.
